### 1.2 Tractability insight (recon) → drives the harness engineering requirements
- Within the novel-at-T frame `ac_lit_asof ∈ {0,1}` **by definition**, so the novelty-penalty term
  `−w·log1p(ac_lit_asof)` barely varies (0 or −0.69); within-frame Wayfinder score is driven by
  `min(z(ab),z(bc)) + β·z(effect)`, all as-of-T.
- **Referee verdict is local/FREE** (cached T1–T4 tables via `load_referee_data()`); no network.
- **Network cost** = `ac_lit_asof(2016)` for all 47,220 (frame definition) + `ac_lit_now` for frame members
  (positive labels) + `ab_asof` per-gene (3,935) + `bc_asof` per-disease (12). Dominant = ~47k asof + ~frame-size
  now queries → **multi-hour job**. The spike ran serial (~1.1 s/call ⇒ ~24 h cold for the full frame). Plan v3
  did **not** spec concurrency. **Therefore the harness MUST be: cached + resumable + checkpointed + small
  concurrent pool (≤8) + polite backoff**, with a `--subset N` mode for the blinded gate. This is the one
  material engineering addition beyond plan v3, recorded here as a deliberate, auditable decision.
