## (B) PLAN FINDINGS

- P0 — §1 decision rule: the rule can call "DOWN" from ARM A + ARM C even if both are observational/reversal correlations and ARM B is null/opposite. Fix: require ARM B to be concordant or, at minimum, downgrade any A+C-only call to "association-backed, not perturbation-backed"; require an explicit effect-size threshold, FDR threshold, and direction consistency across primary/backup datasets.

- P0 — §1 ARM A / §3 / §5 bulk attribution: lesional bulk-skin NAB2 up could reflect keratinocyte abundance, infiltrating immune cells, stromal shifts, or biopsy composition rather than per-cell transcriptional regulation. Fix: include deconvolution/cell-marker adjustment for ARM A and require ARM D pseudobulk by cell type before interpreting topical/T-cell direction.

- P0 — §3.1 NAB2 row resolution: RNA-seq rows are mixed across symbol and Ensembl formats, while array replacements can have multiple NAB2 probes. Fix: define a deterministic resolver: Ensembl `ENSG00000166886`, symbol `NAB2`, Entrez `4665`; collapse duplicate/probe rows by pre-specified rule, report all probes if discordant, and never pick the most favorable row.

- P1 — §1 ARM A Th2 score is underspecified: IL4/IL5 are often low/dropout in skin RNA-seq; mixing cytokines, chemokines, and TFs into one mean-Z score may overweight cell abundance. Fix: predefine included genes, missing-gene handling, min-expression filters, z-scoring scope, and sensitivity analyses excluding low-detection cytokines and using established type-2 modules.

- P1 — §2 / §3 GSE330551 normalization and meta-set claim: GEO exposes n=174 counts and metadata, while the plan text leans on ">1,300 lesional/non-lesional transcriptomes from 30 studies." Fix: either analyze only the downloadable n=174 dataset with transparent normalization, or separately obtain/validate the full meta-set and model study/batch/endotype as covariates.

- P1 — §3 ARM B biological specificity: GSE292848 is ex vivo native skin and GSE282371 is epidermal/keratinocyte equivalents; neither proves NAB2 regulation in CD4 T cells. Fix: label ARM B as skin-cell/epithelial Th2 cytokine response and add a T-cell cytokine-stimulation dataset if the final claim is about T-cell NAB2 direction.

- P1 — §3.6 / §5 STAT6 co-report: co-reporting STAT6 is useful but not sufficient for a cis/locus confound. Fix: add local 12q13.3 checks: NAB2-vs-STAT6 correlation, whether the same direction occurs after STAT6 adjustment, and whether array probes cross-hybridize or RNA-seq multi-mapping is plausible.

- P1 — §4 / §5 treatment reversal: the listed ARM C primary (GSE183953) is PBMC scRNA and not skin; a blood proxy should not have equal vote weight with skin arms. Fix: replace or prioritize with verified skin dupilumab arrays GSE130588 or GSE59294, using GPL570 NAB2 probes and paired/treatment/time modeling; keep PBMC only as supportive context.

- P1 — §1 decision failure mode: UP-modulate vs DOWN-modulate is inferred from expression tracking, but NAB2 could be an induced negative-feedback brake in Th2 inflammation. Fix: separate "tracks Th2" from "pathogenic effector"; require language like "DOWN-modulation hypothesis is consistent with tracking" unless perturbation evidence shows that reducing NAB2 lowers Th2 outputs.

- P2 — §3 statistics: paired designs are present in ARM A/C and biological donor pairing exists in stimulation studies. Fix: specify paired or mixed models by patient/donor where metadata support it; avoid treating repeated biopsies/timepoints as independent.

- P2 — §4 deliverable: acceptance criteria mention p/FDR but not minimum usable data. Fix: add pre-flight gates: processed matrix present, NAB2 row resolved, at least 3 biological units per contrast, metadata sufficient for contrast, and a "no call" path if any primary arm fails.

- P2 — Missing confounds: AD skin datasets may vary by lesion chronicity, treatment washout, age/sex, site, severity, infection/S. aureus burden, and medication history. Fix: extract available covariates and report whether direction is robust or untestable after covariate constraints.
